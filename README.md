# termux-archlinux
Run ArchLinux in Termux.
## Install
`apt update && apt install wget proot -y && wget -c https://github.com/Flytreels/termux-archlinux/raw/master/arch.sh && bash arch.sh`
## Support
 Arch  | Support  |
 ---- | ----- | 
 arm  | √ |
 aarch64  | √ |  
 amd64  | x |  
 i386  | x |
